以下是LinCC（Lin C Compiler）的使用帮助说明：

0.功能介绍
本程序包含三类功能：文件，编译，分析
文件类功能包括打开文件，关闭文件和退出
编译类功能包括编译-生成目标代码和编译-生成中间代码
分析类功能包括展示词法分析，LL1文法展示，预测分析表展示，语法分析展示，语法树展示，语义分析展示


1.主界面：
主界面上方是打开的源代码，下方是操作日志


2. 文件类功能
2.1打开文件：
在文件打开对话栏中打开文件即可。这里推荐打开同目录下的testfile文件夹下的C源程序文件，
打开文件后，文件会显示在主界面上方

2.2关闭文件：
关闭后，文件将会关闭

2.3退出：
退出程序

3.编译类功能
需要先打开文件，才能编译，若未打开文件会给出提示。
3.1编译-生成目标代码：
会生成中间代码（四元式形式）和目标代码（Mips32汇编代码）。其中，中间代码文件的命名为"xxx_Quat.txt"（假设源文件为xxx.c），目标代码文件的命名为"xxx.asm"。
若编译失败，会告知出错原因。

3.2编译-生成中间代码：
仅生成中间代码，命名同上


4.分析类功能
除LL1文法、预测分析表展示功能不需要编译以外，其他功能均要求编译成功。否则会给出相应提示
4.1 词法分析结果
词法分析结果展示形式为 "< 标识符 , 字符串>"，其中标识符以"$"开头，字符串即为源代码中的相应词句
4.2 LL1文法展示
会给出使用的产生式以及所有非终结符的FIRST，FOLLOW集合。
4.3 预测分析表展示
会给出LL1预测分析法中使用的预测分析表。
4.4 语法分析展示
会给出LL1预测分析法的分析过程和语义分析中使用的产生式
4.5 语法树展示
会给出使用的语法树。若语法树较为庞大，则不会显示完全，但可以通过滑轮或者是方向键进行移动。
注意，点击后可能会弹出黑框，此为正常现象，请不要手动关闭黑框！！！
4.5 语义分析展示
会给出生成的四元式，变量表和符号表
